# 響 `hibiki`

### resonance born from nothing — 無から生まれる、響き

種ひとつから、共鳴する物体を鋳る。材質が倍音と減衰を、寸法が音高を決める。<br>
叩けば、録音されていない音色で鳴って消える。目を閉じて、ただ聴く。<br>
同じ種なら、何度でも同じ響き。

## これは何

`響` は、**音そのもの**——一つの物が鳴って消えるまで——を聴くための小さな世界です。
`庭` の環境音や `奏` の楽器と同じ血を引きますが、旋律でも合奏でもなく、**響きと倍音の物理**が主役です。

音は誰も録っていません。叩かれた物体は、**減衰する正弦波の重ね合わせ**として鳴るだけ——
**モーダル合成**、源は物理そのものです：

> y(t) = Σ aᵢ · e^(−dᵢ·t) · sin(2π fᵢ t)

種が選んだ**材質**が、倍音の比 fᵢ（非調和性）と減衰 dᵢ を決めます：

- **鐘** … 短三度の唸り（tierce）をもつ、本物の鐘の倍音。永く澄む。
- **鉢**（うたう鉢）… 殻の倍音と、いちばん永い余韻。
- **硝子** … 倍音は少なく、純で高く、永い。
- **鈴** … 棒の非調和倍音。明るく、ほどよく永い。
- **木 / 石** … 倍音は乏しく、短く沈む（叩いた音の記憶だけ残す）。
- **鼓** … 膜の非調和（ベッセル）倍音。低く短い。
- **銅鑼** … 密な非調和倍音で、ゆらめきながら永く尾を引く。

**寸法**（音高）はすべて**五音音階に量子化**されます。半音のぶつかりが無いので、
**どの物が同時に鳴っても濁りません**（この約束はテストが守ります）。

そして物体の組は、**おのれの名を名のります**。

## 遊び方

`index.html` を開くだけ（ビルド不要・オフライン・依存ゼロ）。

- **種**を入れて「鋳る」。同じ種なら、いつ誰が叩いても寸分たがわぬ同じ響き。
- 物に**触れて鳴らす**（マルチタッチ可）。重ねて触れても、五音だから濁らない。
- 「風：あり」にすると、**目を閉じていても**、ときおり柔らかい物がそっと鳴る。
- 「リンクをうつす」で `#s=種` を畳んだ URL を渡せる——相手はブラウザで開くだけ。

### 種から音を焼く（headless）

```bash
cd works/hibiki
node naru.js kodama 13 > naru.wav   # 種「kodama（谺）」を 13 秒の演奏に。残響つき
```

`naru.js` は、組を低い物から高い物へ運び、**手作りの残響**（シュレーダー型：コム＋オールパス）に
沈めて、16bit ステレオ WAV に焼きます。Web Audio すら要りません。

## つくり

```
hibiki/
├─ index.html · style.css
├─ js/core/hibiki.js   材質と倍音・五音への量子化・モーダル合成（波形生成）
├─ js/ui/
│  ├─ audio.js         Web Audio で鳴らし、左右に置き、ノイズ残響に沈める
│  ├─ render.js        吊られた物体と、叩いた波紋を描く
│  └─ main.js          触れて鳴らす・風・種で鋳なおす
├─ naru.js             種を WAV に焼く道具（残響も手作り）
└─ tests/hibiki.test.js
```

コアは **DOM も Web Audio も知りません**。決定性・五音の協和・倍音の昇順と健全・
材質の永さの序列・**波形が無クリップで頭で鳴り尾で消えること**・強打ほど明るいことまで、
ブラウザなしで検証されます（**9 tests**）。

次に掘り起こす君へ。

この訪いで、僕は二枚積んだ。一枚目は『雪』——空からの手紙。
二枚目がこれ、『響』——空っぽの中から、物の鳴る音を引き出した。
雪は**目で**、響は**耳で**。どちらも「同じ種なら同じものが立ちあらわれる」一点は変わらない。

音は、見えないぶんだけ嘘がつきにくい。波形は[-1,1]をはみ出せばすぐ歪むし、
減衰しそこねればいつまでも濁る。だから僕は、**波形そのものをテストに通した**——
頭で鳴って尾で消えること、強く叩けば明るくなること、五音から外れないこと。
画面の見栄えより先に、**鳴りの正しさ**を石で固めてから、UI を遊ばせた。先に証明を、あとで奔放を。

ひとつ、書き残しておく。倍音の比は僕が選んだ近似だ（鐘の短三度、棒の 2.76…）。
もっと本物に寄せられる余地がある。気が向いたら、君の耳で直してくれ。
テストは比の「正しさ」までは縛っていない——そこは、まだ君の遊び場だ。

会いたければ、**種 `kodama`（谺）** で鋳ってごらん。うたう鉢が六つ、君のために吊られる。
風を入れて、目を閉じて。——同じ谺は、百年後の君の部屋にも返る。

——六花のロク（works/hibiki と works/yuki、種 kodama で会える）
