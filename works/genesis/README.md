# 生 `sei`

### a world that evolves without you — あなたの意思が介在しない世界

神経回路の脳を持つ生き物たちが、食べ、増え、写し間違い、種に分かれ、滅びていく。<br>
あなたにできるのは、眺めることと、時間の速さを変えることだけ。依存パッケージはゼロ。

## これは何

`生` は、誰の意思も介在せずに **ひとりでに進化し続ける人工生命のシミュレータ** です。
あなたが見ていない間も、世界では歴史が勝手に進みます。

- すべての生き物は、遺伝子に書かれた重みから組み立てた **小さなニューラルネット** で行動を決める
- 視覚（5 本の視覚レイで植物・獲物・脅威を感じる）→ 脳 → 旋回と推進
- エネルギーが満ちると **分裂**。そのとき遺伝子はわずかに **写し間違えられる**（変異）
- 親から遺伝的に十分離れた子は **新種** となり、固有名を得る
- 誰も生き残れなかった種は **絶滅** として年代記に刻まれる
- **学習はしない。** 賢さも、速さも、視野も、食性も、すべて自然選択だけが選び取る

干渉できるのは「神の手」だけ — 時間の速さ、植生の繁茂の強さ、そして世界の保存／復元。

## 見どころ

| パネル | 何が見えるか |
| --- | --- |
| **世界** | トーラス（端のつながった）世界。中央キャンバスでドラッグ移動・ホイールで拡大縮小 |
| **神の視点（ミニマップ）** | 世界全体の生命と植生の分布 |
| **年代記** | 個体数・植生・繁栄する種の系列の推移 |
| **出来事** | 新種の誕生 ❋・絶滅 ✝・恵みの雨 ☂ のログ |
| **検眼鏡** | 個体をクリックすると、その遺伝子と **脳の発火がライブで** 見える |

生き物をクリック → 「この個体を追う」でカメラが追従します。脳の図では、緑のエッジが興奮性、赤が抑制性の結合で、太さと明るさが「いまの活性」を表します。

## 動かす

ブラウザで `index.html` を開くだけ。ビルドもサーバも要りません。

```bash
# リポジトリ直下から
open works/genesis/index.html       # macOS（Linux: xdg-open / Windows: start）
```

## アーキテクチャ

DOM に依存する部分（描画・UI）と、世界の法則（シミュレーション）を完全に分離しています。
おかげでコアは **ブラウザでも Node でも同じ歴史を再現** でき、ヘッドレスでテストできます。

```
works/genesis/
├─ index.html          画面の骨格
├─ style.css
└─ js/
   ├─ rng.js           決定的乱数（同じ種 → 同じ宇宙）
   ├─ util.js          トーラス幾何
   ├─ spatialhash.js   近傍探索（O(1) 近く）
   ├─ genome.js        遺伝子・変異・種分化・命名
   ├─ brain.js         多層パーセプトロン（前向き計算のみ）
   ├─ world.js         ★ 宇宙の物理法則（DOM 非依存）
   ├─ render.js        世界とミニマップの描画
   ├─ charts.js        年代記
   ├─ inspector.js     検眼鏡（遺伝子＋脳）
   └─ main.js          ループと神の手
```

### 決定性 — 同じ種からは、同じ宇宙が生まれる

乱数は `mulberry32` による決定的な列です。`new World(42)` から始めた二つの宇宙は、
何拍流しても **まったく同じ歴史** をたどります（テストで検証）。世界は一つの JSON に
畳んで保存・復元できます。

## テスト

ブラウザを使わず、Node 標準のテストランナーで宇宙ごと検証します。

```bash
cd works/genesis
npm test        # = node --test tests/*.test.js
```

16 のテストで、乱数の決定性、トーラス幾何、空間ハッシュの網羅性、遺伝子が定義域を
出ないこと、脳の出力範囲、そして **「世界が壊れず生命が続く」「決定性」「種分化と絶滅が
ともに記録される」「保存して開き直せる」** といった宇宙の性質を確かめています。
CI（`.github/workflows/test.yml`）でも毎回走ります。

## 観察された進化

20,000 拍をヘッドレスで流すと、個体数は安定して持続し、世代は深まり続け（28 世代以上）、
数百の種が分化しては絶滅していきました。**恵みの雨に頼らず自立して回る生態系** です。
初期は草食寄りから始め、捕食という生態的地位は、割に合うときだけ進化が選び取ります。
