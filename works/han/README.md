# 反 `han`

### Reversi, a mind built from nothing — 無から立ち上がる、考える相手

## これは何

`反` は、**リバーシ（オセロ）**です。ただの盤ではなく、**無から立ち上がる「考える相手」**つき。
`算`（無から作った計算機）が機械なら、こちらは**無から作った、盤上の知性**。

- **ネガマックス＋α-β枝刈り**で先を読む
- **位置評価**：隅は宝（+120）、隅の隣は罠（−40）。**機動力**（打てる手の多さ）も読む
- **終盤は読み切る**：空きが少なくなると、最後の一マスまで完全に読んで石差を最大化する
- **乱数を使わない**——同じ局面からは、いつも同じ最善手（決定的）

### 強さは、数で確かめられる

弱いと言い張ることも、強いと誇ることもできません。**勝負の数**が答えます。

```
node han.js --bench 60        AI(深さ4) 対 ランダム 60局 → 勝ち 60 / 引分 0 / 負け 0
node han.js                   AI 対 AI の決定的な一局（深いほうが、浅いほうに勝ち越す）
```

- コアは **DOM もネットワークも知らない**。同じ局面からは寸分たがわず同じ手
- 規則（合法手・反転・パス・終局）・勝者・**対ランダム無敗**・**深い読みほど強い**・終盤の読み切りまで
  **ヘッドレス検証・10 tests**

## 遊び方

**ブラウザ**：`index.html` を開く。あなたは黒（先手）。打てる場所に小さな印が出る——置けば相手が考えて返す。
「強さ」を **やさしい / ふつう / つよい / 鬼** から、手番も黒白から選べる。

**ターミナル**：
```bash
node han.js               # AI 対 AI の一局を清書（棋譜と盤）
node han.js --depth 6     # 読みの深さを変える
node han.js --bench 100   # ランダム相手の勝率
```
