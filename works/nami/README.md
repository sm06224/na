## これは何

`波` は、**触れて眺めるだけ**の水面です。`籤` が理屈っぽい一作だったので、その揺り戻し——
理屈はさておき、ただきれいで、ただ気持ちいいもの。`庭`（流れ場）・`蛍`（六月の光）の血を引きます。

波形は誰も描いていません。水面の高さを格子に持ち、**離散化した波動方程式**を一歩ずつ進めるだけ：

```
h' = (2h − h_prev) + c²·∇²h        ∇² は四近傍ラプラシアン
h' *= damp                          ゆっくり凪いでいく減衰
```

- 岸は**反射壁**（格子の外の隣は自分自身）。だから波紋は縁で跳ね返り、互いに**干渉**する
- 描くときは、各セルの**傾き**で空のグラデを**屈折**させて光の帯（caustics）にし、
  稜（盛り上がり）に**きらめき**（specular）を載せる
- **長く押すほど、大きく深い波**になる（押した長さで振幅と半径がふくらむ）
- 一雫ごとに、雑音と発振から手づくりの水音が鳴る——本物の水滴のように少し上がって澄み、
  **大波ほど低く深く**。自前のリバーブ（帰還する遅延）を通して、池に**響く**。
  押しているあいだは「ためる」含み音が、押すほど深くなる

物理シミュレータのつもりはありません。`生`（神経）・`雷`（電位の場）が形をひとりでに生んだのと
同じ系譜——**場の自己組織化**を、手のひらの池にしたものです。

- コアは DOM を知らない。**Node の中でも、同じ波が立つ**（ヘッドレス・8 tests）
- 対称性・安定性（発散しない）・減衰（やがて凪ぐ）・**重ね合わせ（線形性）**まで、テストが守る

## 読み方

`index.html` を開いて「水にふれる」。

- **触れる**と波紋がひとつ生まれる。**長く押すほど**大きく深い波に（押した長さで決まる）
- **なぞる**と航跡になり、**複数の指**でも触れる
- **待つ**と、雨がぽつりぽつりと落ちはじめる（触れているあいだは降らない）
- 「凪にする」で水面をまっさらに、「音を消す」で雫の音を止められる
