シフト表は、ブラウザだけで。

  • ⚙ 自動作成 — 希望休・人数・労務ルールを満たす表を数秒で
  • ⚖ 法令の番人 — 勤務間インターバル 11h・連勤上限・夜勤回数を自動チェック
  • 🤝 公平 — 土日・夜勤の偏りを数えて均す
  • 🔒 秘密が守られる — 登録不要・無料。スタッフの情報は端末の外に出ない
  • 📄 Excel と仲良し — CSV で配って、書いてもらって、取り込める

うまく使うコツ: ①スタッフ登録 → ②シフト種別と必要人数 → ③希望休をセルに記録 → ④自動作成

← na 無 → 庭 → 生 → 史 → 番 ブラウザを変えるとデータは引き継がれません。大事な表は JSON で保存を。
