## これは何

`層` は、**深い時間を断面で読む**ための小さな世界です。`蛍` の光や `波` の水面と同じ
血を引く——理屈はコアにあり、画面には静けさだけがある。

一年ごとに、土が一枚ずつ降り積もります。誰も縞模様を描いていません。
**気候（いくつかの正弦波の重ね合わせ）と種から生まれる揺らぎ**が、その年の堆積を決めるだけ：

- **雨の年**は厚く、運搬の勢いが強いので**粗い**（礫・砂）。
- **乾いた年**は薄く、静かに**細かい**（泥・粘土）。
- ごくまれに **大水・火山灰・旱魃・地震・繁茂** が訪れ、ひと筋の際立った縞となって残ります。

積もった層は、**己の上に載る重み**でゆっくり圧されます（圧密）。だから下にある古い層ほど
薄く締まっている。最後に侵食が崖を切ると、地表（今）から岩盤（最古）まで、
時間がそのまま縞の重なりとして立ちあらわれます。

右の物差しは **深さ＝時間**。縞に触れれば、その年と粒度がわかります。

## 遊び方

`index.html` を開くだけ（ビルド不要・オフライン・依存ゼロ）。

- **種**を入れて「掘る」。同じ種なら、いつでも寸分たがわぬ同じ大地。
- 「別の地を」で、見知らぬ土地の断面を掘り出す。
- **歳月**のつまみで、刻む年数を変える（深い時間ほど、際立つ縞も増える）。
- 崖の縞に触れて、「地の年代記」を読む。

## コア（`js/core/strata.js`）

UI を持たない純関数だけ。ブラウザも乱数の外部依存もいらない。

| 関数 | はたらき |
|---|---|
| `deposit(seed, years)` | 種から years 年ぶんの層（古い順）を積む |
| `compact(layers)` | 上載の重みで圧密する（深いほど薄い・単調） |
| `cliff(seed, years, rows)` | 侵食面を rows 段に写す（上が新しく下が古い） |
| `readRecord(seed, years)` | 刻まれた出来事を新しい順に読み上げる |
| `renderText(...)` | 端末でも崖を文字で確かめる |

```bash
cd works/sou
node --test tests/*.test.js
```

検証していること：**種は決定的**（同じ種なら同じ大地）／一年に一枚・古い順・厚みは正で有限／
**圧密は単調**（深い層ほど薄く、元より薄く、なお正）／長い歳月には出来事が必ず幾本か刻まれ再現する／
崖は**地表（最新）から岩盤（最古）まで貫き**、全段が実在の層に対応する。
