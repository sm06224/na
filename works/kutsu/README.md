# 窟 `kutsu`

### a labyrinth no one designs — 誰も設計しない迷宮

種をひとつ蒔くと、洞窟も通路も、魔物も宝も、そして潜行の年代記まで生まれる。<br>
同じ種からは、同じ迷宮。死は終わり——でも種は残る。依存パッケージはゼロ。

</div>

---

## これは何

`窟` は、**種ひとつから一揃いのローグライクが生まれる**作品です。`生`（人工生命）・
`史`（歴史）・`星`（夜空）と同じ「ひとりでに生まれる」系譜の、地の底の章。

- **誰も部屋を描かない**：部屋＋通路・セルオートマトンの洞窟・BSP・迷路が深さに応じて
  選ばれ、繋がれ、扉・罠・水・草・像・泉・階段が据えられる
- **誰も魔物を置かない**：深さ帯ごとに棲むものが変わり、群れは固まって湧き、まれにヌシが待つ
- すべては種の純粋関数。**同じ種からは、一マスもちがわない同じ迷宮**——リンク（`#s=種`）で渡せる
- 死は永遠（ローグライク）。けれど種は残るので、**同じ窟へ何度でも**挑める
- コアは DOM を知らない。**Node の中でも、同じ窟が掘れて、同じ潜行が回る**（ヘッドレス・56 tests）

## 遊び方

`index.html` を開き、**冒険者の型**（戦士・魔術師・盗賊・狩人）を選び、種を入れて「潜る」
（空ならおまかせ）。第 15 階の**護符**を取り、上り階段から地上へ戻れば生還。死ねば終わり——（保存した潜行は「続きから」で再開できる）
でも種は残り、過ぎた潜行は表紙に記録される。

- **移動**：矢印 / `hjkl` / `yubn`（斜め）/ テンキー　・**待つ** `.`
- **拾う** `g`・**買う** `p`（店）・**降りる** `>`・**上る** `<`・**調べる** `x`／`s`（隠し扉・罠）
- **薬** `q`・**巻物** `r`・**杖** `z`（→ 向き）・**技** `a`（型の特技・気力を消費）・**装備** `w`・**投げる** `t`・**食べる** `f`・**置く** `d`
- **持ち物** `i`・**装備** `e`・**人物** `@`・**鑑識帳** `L`・**音** `M`・**ヘルプ** `?`
- 青い薬が回復とは限らない——**未鑑定の品は見た目だけ**。使うか鑑定で正体が知れる（見た目は潜行ごとに種で入れ替わる）
- まれに**特殊部屋**（宝物庫・牢・祠・書庫・**店**…）や**遺物**（名のある唯一の品）に出会う。
  店主は中立だが、盗めば牙を剥く

## しくみ（コアは DOM 非依存）

```
works/kutsu/js/
├─ core/                          ★ ぜんぶ Node で動く
│  ├─ rng util tile level         決定的乱数・幾何・タイル・階層と覚え
│  ├─ fov pathfind                視界（シャドウキャスト）・A*・匂い地図
│  ├─ gen/                        部屋/洞窟/BSP/迷路・連結・装飾・特殊部屋(vault)・建て分け
│  ├─ entity board factory        実体・盤・鋳型
│  ├─ item itemdb inventory       品物（80種超）・鞄・装備・呪い・付呪
│  ├─ identify                    見た目の入れ替え
│  ├─ monsterdb combat status ai  魔物（50体超）・戦い・状態異常・知恵
│  ├─ effects                     薬14・巻物12・杖10 の効能
│  ├─ player spawn chronicle      成長と空腹・配置・年代記と墓碑銘
│  └─ game actions                エンジン（エネルギー制の手番）と一手
└─ ui/  render screens main       格子の描画・重なる札・操作
```

```bash
cd works/kutsu
node --test tests/*.test.js   # 100 tests（決定性・連結・視界・経路・自動潜行・戦闘・効果・型と技・遺物・店・保存と再開…）
```

死ぬと、この潜行の**墓碑銘**が立つ（`史` の血を引く年代記から）。種は墓標に刻まれる
——`#s=種` を誰かに渡せば、まったく同じ窟へ送り込める。
