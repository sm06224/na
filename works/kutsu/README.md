# 窟 `kutsu`

### a labyrinth no one designs — 誰も設計しない迷宮

種をひとつ蒔くと、洞窟も通路も、魔物も宝も、そして潜行の年代記まで生まれる。<br>
同じ種からは、同じ迷宮。依存パッケージはゼロ。

</div>

---

## これは何

`窟` は、**種ひとつから一揃いのローグライクが生まれる**作品です。`生`（人工生命）・
`史`（歴史）・`星`（夜空）と同じ「ひとりでに生まれる」系譜の、地の底の章。

- **誰も部屋を描かない**：部屋＋通路・セルオートマトンの洞窟・BSP・迷路——複数の彫り方が
  深さに応じて選ばれ、繋がれ、扉・罠・水・草・像・階段が据えられる
- すべては種の純粋関数。**同じ種からは、一マスもちがわない同じ迷宮**——リンクで渡せる
- コアは DOM を知らない。**Node の中でも、同じ窟が掘れる**（ヘッドレス検証）

## いま出来ていること（作りかけ・段階的に育ちます）

この作品は大きいので、何回かに分けて育てています。今あるのは **空間のコア**と
**潜行のシミュレーション**（魔物・戦い・品物・効果・年代記）——どれもブラウザ無しで
動き、自動の潜る者が窟を降りきれることをテストで確かめています：

```
works/kutsu/js/core/
├─ entity / board / factory   実体・盤・鋳型
├─ item / itemdb / inventory  品物（武器・防具・薬・巻物・杖・指輪・食料）と鞄・装備
├─ identify                   見た目の入れ替え（青い薬が回復とは限らない）
├─ monsterdb                  魔物の名鑑（深さ帯・群れ・遠隔・呪文・盗み・召喚…）
├─ combat / status            命中と傷・会心・毒や吸い、状態異常
├─ ai                         眠り・追跡・遠隔・臆病・盗み・擬態
├─ effects                    薬・巻物・杖の効能
├─ player / spawn             成長と空腹・階への配置
├─ chronicle                  記録と墓碑銘（`史` の血を引く）
├─ game / actions             エンジンと一手（エネルギー制の手番）
└─ （空間のコア：rng / util / tile / level / fov / pathfind / gen …）
```

最初にあった空間のコア：

```
works/kutsu/
└─ js/core/
   ├─ rng.js          決定的乱数（さいころ・重み・正規分布・save/restore）
   ├─ util.js         方向・矩形・距離・直線・連結成分
   ├─ tile.js         タイルの種類とその性質（歩ける／透ける／掘れる／液体／扉）
   ├─ level.js        階層の地形と「覚え」（見えた・照らされた・罠を知った）
   ├─ fov.js          視界（再帰的シャドウキャスト）
   ├─ pathfind.js     A* と ダイクストラ地図（魔物が獲物へ寄る／逃げる）
   └─ gen/            ダンジョン生成
      ├─ carve.js     床を刻む低レベルの道具
      ├─ rooms.js     部屋と通路
      ├─ cave.js      セルオートマトンの洞窟
      ├─ bsp.js       BSP（城のような階）
      ├─ maze.js      迷路（編んで回遊できる窟に）
      ├─ connect.js   連結の保証
      ├─ decorate.js  扉・罠・水・草・像・階段
      └─ build.js     深さに応じて建てるまとめ
```

これから、魔物・戦い・品物・魔法・潜行のループ・年代記・ブラウザの画面が加わります。

```bash
cd works/kutsu
node --test tests/*.test.js   # 決定性・連結・視界・経路・生成の検証
```
