# 計 `kei`

### a calculator you can read — 書いた式の右に、答えが出る

プレーンテキストのノート。各行が式で、単位・通貨・%・前行参照を解釈する。<br>
為替のような嘘はつかない。依存パッケージはゼロ。データは端末から出ない。

</div>

---

## これは何

`計` は、**読める電卓**です。メモ帳に書くように計算を綴ると、各行の右に答えが並ぶ。
`算`（無から作った 16bit の計算機／VM）が「機械」なら、こちらは**人のための、ことばに近い電卓**。

```
新幹線 = 13320 円 * 2
宿     = 2 泊 * 8500 円 * 2
小計   = sum                 ← 上のかたまりを合算
予備   = 小計 + 15%
ひとり = 予備 / 2
```

### できること

- **単位**：`km kg h GB …`（日本語の `時間 日 分 秒 週` も）。混ぜても割っても、**次元**がひとりでに正しくなる。
  ```
  100 km / 2 h        → 50 km/h        （複合単位は割り算からおのずと）
  2 TB / 50 MB/s in min → 666.67 min
  5 km in mi          → 3.106856 mi
  ```
- **数え単位**：`区間 個 人 泊 …` のような**知らない語は、そのまま単位になって運ばれる**。
  だから割り算で `区間/日` が生まれ、単価×数では約分される。
  ```
  24 時間/日 / (8 時間/区間) → 3 区間/日
  500 円/個 * 12 個          → 6,000 JPY      （個 が約分される）
  ```
- **組み立て単位**：`N Pa J W Hz V Ω C L kWh cal …`（SI 由来）も次元で表すから、計算と変換がおのずと合う。
  ```
  2 kg * 9.8 m/s^2 in N → 19.6 N
  100 W * 1 h in kWh    → 0.1 kWh
  1 atm in hPa          → 1,013.25 hPa
  ```
  `^` は単位ひとつに掛かる（`5 m^2 → 5 m²`、`9.8 m/s^2 → 9.8 m/s²`）。
- **SI 接頭辞**：どの単位にも前につく（`5 µm`・`500 ns`・`2 GHz`・`3 mΩ`・`10 cL`…）。
- **温度**：`°C ↔ °F ↔ K`（アフィン変換）。`180 °C in °F → 356 °F`。
- **定数と関数**：`pi e tau`、`sin cos tan asin…`（`sin(30 deg)`）、`sqrt round min max avg abs ln log exp mod hypot`。
- **通貨**：`円 ¥ $ USD …`。**それぞれ独立した次元**だから、`USD + JPY` は足し合わせを拒む。
  **為替レートは持たない**——だから決して嘘をつかない（`100 USD in JPY` は、できないと正直に断る）。
- **パーセント**：`80 + 8%`（=86.4）、`20% of 80`（=16）、`1000 円 + 8%`。
- **変数と前行参照**：`x = …`、`prev`（前の答え）、`sum`/`total`（上のかたまりの合計）、`line 3`。
- **関数**：`sqrt round floor ceil abs min max avg ln log`。
- **散文にやさしい**：見出しやメモの行は黙って素通り。けれど演算子つきの**本当の書き間違いは、ちゃんと知らせる**。

### 作法

> **何も無いところに、尽きせぬ計算が宿る。** 嘘の数字（持っていないレート）は、出さない。

- コアは **DOM もネットワークも知らない**。`run(ノート文字列)` が行ごとの答えを返す。
- 同じノートは**寸分たがわず同じ答え**（決定的）。リンク（`#n=…`）で同じノートを分かち合える。
- 値は「基準単位での大きさ＋次元＋表示単位」を背負う**量（quantity）**。

## 使い方

**ブラウザ**：`index.html` を開く。左に書くと、右に答え。例（旅費・単位・暮らし）はボタンから。

**ターミナル**（成果物をそのまま）：
```bash
node kei.js examples/旅費.kei      # ノートを清書して計算
echo "2 TB / 50 MB/s in min" | node kei.js
```

## アーキテクチャ

```
works/kei/
├─ index.html · style.css        左に式・右に答えのノート（行は一対一でそろう）
├─ kei.js                        CLI（ファイル／パイプを清書して計算）
├─ examples/*.kei                旅費・単位・暮らし
└─ js/
   ├─ core/                      ★ DOM 非依存
   │  ├─ value.js                量（次元つき）・単位表・通貨・表示
   │  └─ kei.js                  字句 → 構文(Pratt) → 評価 ・ run(ノート)
   └─ ui/main.js                 textarea に書くたび右へ描き、行をそろえる
```

```bash
cd works/kei
node --test tests/*.test.js   # 12 tests（四則・桁・単位・複合・通貨・%・関数・参照・散文・決定性）
```
