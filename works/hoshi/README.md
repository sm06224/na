# 星 `hoshi`

### a sky that names itself — 誰も設計しない夜空

## これは何

`星` は、**誰も設計しないのに、星座とその名と神話が、ひとりでに生まれる**夜空です。

1. 種ひとつから、600 の星が撒かれる（位置・明るさ・色温度）
2. 明るい星どうしが、近ければ**おのずと結ばれて星座になる**（最小全域木で枝を張る）
3. それぞれの星座が、**自分で名のる**（開いた音だけの、やわらかな架空語）
4. かたち（細長い・まるい・広がる）と主星の色から、**由来＝神話が書かれる**
5. 空でいちばん明るい星が、**一番星**として最初に灯る

地図アプリのように本物の星座を写すのではありません。`言`（言葉）・`歌`（旋律）・
`史`（歴史）が、誰の手も借りずに生まれたのと同じ系譜——**「ひとりでに名づけ、
物語る」ものを、夜空にひろげた**作品です。

- すべては種の純粋関数。**同じ種からは、一星も一文字もちがわない同じ空**がひらく
- だから空は**リンクで渡せる**（`#s=種`）。誰かと同じ夜空を見上げられる
- コアは DOM を知らない。**Node の中でも、同じ星が瞬く**（ヘッドレス・9 tests）

## 読み方

`index.html` を開いて「夜をひらく」。

- 指でなぞって**星座にふれる**と、その名と由来（神話）と主星が読める
- 金の光輪が**一番星**。「一番星へ」で寄れる
- 「べつの空をみる」で、まだ誰も見ていない種の空がひらく
- 「この空を送る」で、いまの夜空をそのままリンクにして渡せる
