# 針 `hari`

### a needle that remembers the way back — 帰り道を覚えている羅針盤

駐車場で。フェスのテントで。はじめての街の宿で。<br>
針を刺しておけば、あとは**矢印と距離だけ**が連れ戻す。地図は出ない。だから地下でも山でも動く。

</div>

---

## これは何

`針` は、**「あの場所へ戻る」だけを引き受ける**道具です。三番目の実用品。

1. 車を停めたら **「ここに針を刺す」** をワンタップ
2. 帰るとき開くと、**矢印が車の方を指し、距離と徒歩何分**かが出る
3. それだけ

地図アプリと逆の発想です。地図を出さないから、**地図タイルのダウンロードがない＝
完全オフライン**で動く。電波のない立体駐車場・山・海外で、いちばん効きます。
登録なし・サーバーなし・**場所のデータはこの端末から一歩も出ません**。

### 待ち合わせ（サーバーのない位置共有）

「送る」を押すと、座標を **URL の中に畳んだリンク**ができます
（`#s=…` — サーバーには何も保存されない。リンクそのものが場所）。
LINE でもメールでも貼って送れば、相手は**ブラウザで開くだけ**。
アプリのインストールも登録もなしに、相手の画面で針が立ち、
あなたのいる場所まで矢印が案内します。雑踏のフェス会場・花火大会・
スキー場の集合に。

## スマホでこそ

| 端末の力 | 針での使われ方 |
| --- | --- |
| GPS | 刺すときは**数秒ぶんの測位を精度で重みづけ平均**して、いちばん確かな一点に刺す |
| 磁気センサー | 矢印が実際の向きに追従。**国土地理院の磁気偏角近似式（2020.0年値）で磁北→真北を補正** |
| 振動 | 🔔 をオンにすると、**正しい方向を向くほど速く脈打つ**。夜・人混みで画面を見ずに歩ける |
| 共有 | OS の共有シートから LINE・メールへそのまま |
| Wake Lock | 歩いているあいだ画面が落ちない |
| PWA | ホーム画面に入れれば全画面・オフライン起動 |

PC でも開けますが（方位センサーが無いので「上を北として、北東のほうです」と
ことばで補う）、この道具はスマホで使ってこそです。

## 正直さについて

- 測位の精度（±何m）は常に表示します。粗いときは粗いと言います
- 目的地の精度より近づいたら「**もう着いています。あたりを見まわして**」に切り替わります。
  GPS は最後の数 m を案内できないからです
- 方位センサーが断られたら、矢印を「北上の地図」として読む方式に黙って切り替わります

## アーキテクチャ

```
works/hari/
├─ index.html · style.css
├─ manifest.webmanifest · sw.js · icon.svg · icon-180.png   PWA とオフラインの備え
└─ js/
   ├─ core/              ★ DOM 非依存 — 針の数学はぜんぶここ（23 tests）
   │  ├─ geo.js          距離（haversine）・方位角・磁気偏角・真北補正・
   │  │                   向きの平滑化・測位の加重平均・距離のことば・振動の脈
   │  └─ spots.js        針の覚え書きと、URL に座標を畳む分かち合いの符号
   └─ ui/
      ├─ sensors.js      GPS・方位（iOS の許可と webkitCompassHeading / absolute alpha の癖）・振動・Wake Lock
      ├─ needle.js       文字盤 — 継ぎ目をまたいでもなめらかな針
      └─ main.js         刺す・歩く・送る・受け取るの流れ
```

```bash
cd works/hari
node --test tests/*.test.js   # 23 tests（球面三角・偏角・符号化・覚え書き）
```

磁気偏角の根拠：[国土地理院 地磁気値（2020.0年値）](https://vldb.gsi.go.jp/sokuchi/geomag/menu_04/index.html)の
近似式。テストは東京 7°40′・札幌 9°30′・那覇 5°00′ の観測値に対して照合しています。
日本域の外では補正せず、そのぶん針は数度訛ります（矢印を辿る用途では困らない範囲）。
