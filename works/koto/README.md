# 言 `koto`

### a language that invents itself — 無から、言葉が生まれる世界

群れが音をさぐり、通じた音だけが約束になる。<br>
単語は発生し、訛り、方言に分かれ、意味がずれ、死語になる。依存パッケージはゼロ。

</div>

---

## これは何

`言` は、**誰も設計しない言語が、ひとりでに育つ**のを眺める作品です。

はじめ、群れ（deme）はひとつも言葉を持ちません。生きるうちに「捕食者がいる」「餌がある」
「こっちへ来い」を伝える必要が生じ、その場で音節を組んで**新しい単語を発明**します。
聞いて通じれば、その語は約束として強まる。通じなければ、別の言い方が試され、
あるいは**意味が近い概念へ滲み出して**いく。

- すべての語は音節 id の列。子音 × 母音の在庫から、群ごとの**音素プロファイル**が訛りを生む
- 話して**通じた語は強化、使われない語は減衰**して死語になる（語彙の自然淘汰）
- よく通じ合う群は栄え、言葉の通じない群は縮む — **言語が生存と結びついている**
- 群が大きくなると**分裂**し、子は親の語彙を受け継ぐが、独立に訛って**方言**になる
- 近くの群とは**接触**し、弱い語は相手の語に置き換わる（**借用・収束**）
- 「捕食者の季節」のような**世界事件**が、特定の概念の語を急いで磨かせる

辞書も語源年表も、**勝手に書かれていきます**。

> seed 20260612、600 年後の「ビヤト語」辞書：
> 捕食者 = ギ ／ 餌 = ノヤ ／ 水 = ロデ ／ 集まれ = レビナ ／ 求愛 = ハホ ／ 仲間 = ヂム

## なぜこれを作ったか

言語は誰かが設計したものではなく、意思疎通の必要に迫られた個体の間で**勝手に結晶化**
したものです。これを「眺めて美しい作品」として誰でもブラウザで見られる形にした例は、
ほとんどありません（研究の世界の「シグナリングゲーム」「創発コミュニケーション」を、
鑑賞物に翻訳したもの）。

`生`（進化）と `史`（年代記）で培ったエンジンが両方流用でき、しかも出力が
*言葉そのもの* になる——作品集の到達点のひとつです。

## 読み方

| パネル | 何が見えるか |
| --- | --- |
| **地図** | 円 = 言語の群、大きさ = 人口、色 = 系統。線 = 親子（方言の分岐）と接触。発話は概念の色の波紋 |
| **辞書** | 群をクリックすると、その言語の概念ごとの優勢語・競合する同義語・定着度が読める |
| **言語史 / 全史** | 語の誕生 ✦・死 †・意味のずれ ↝・方言の分岐 ⋔・借用 ⇄ |
| **年表** | 語彙数（白）・群の数（紫）・相互理解度（緑）の推移 |

群を選んで「辞書を保存」すると、その言語の辞書をテキストで書き出せます。
速さは ⏸ / ×1 / ×4 / ×12。世界はまるごと JSON に保存・復元できます。

```

└─ js/
   ├─ core/              ★ DOM 非依存 — Node でも同じ言語史が流れる

   │  ├─ meaning.js      概念の在庫と意味的な隣接関係
   │  ├─ lexicon.js      語彙：造語・選択・強化・減衰・同義語淘汰・相互理解度

   │  └─ world.js        ★ 群・発話・分裂・接触・世界事件を束ねる
   └─ ui/
      ├─ render.js       地図と系統の描画
      ├─ panels.js       辞書・フィード・全史・年表グラフ
      └─ main.js         ループと読者のまなざし
```

### 言葉が生死する仕組み（命名ゲームの拡張）

1 回の発話で、概念を選び（世界的必要度で重みづけ）、語彙から語を選ぶ。
通じる確率は語の**定着度**で決まる。通じれば強化、通じなければ —
新しい言い方の発明 ／ 近接概念への**意味のずれ** ／ 言い直しの中の**音変化** の
いずれかが起こる。語は毎年わずかに減衰し、使われぬものは死語になる。
群の分裂で語彙が複製され、独立に訛って方言となり、接触で借用が起こる。

**決定的**：同じ種からは一字一句同じ言語史が生まれる（テストで全文比較）。

## テスト

```bash
cd works/koto
npm test        # = node --test tests/*.test.js
```

20 のテストが、音節の往復・音変化・語形距離・語彙の生死・相互理解度といった部品から、
**「言葉が無から生まれる」「方言が分かれて通じにくくなる」「意味の変化と死語が刻まれる」
「決定性」「保存と復元」** という言語史そのものの性質までを検証します。
CI（`.github/workflows/test.yml`）でも毎回、言語が無から育ちます。
