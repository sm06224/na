# 狐 `kitsune`

### a GPS treasure hunt — スマホひとつの、宝探し

狐が街に的を仕掛け、追手はリンクをひとつ受け取って、<br>
矢印だけを頼りに駆ける。地図は出ない。だから知っている街も、迷宮になる。

</div>

---

## これは何

`狐` は、`針`（帰り道の羅針盤）から生まれた**二人以上で遊ぶ宝探し**です。

- **狐（出題者）** は現地を歩き、いくつかの的（チェックポイント）を仕掛ける。
  名前・ヒント・通過のしるしを添えて、**コースまるごとを一本のリンクに畳む**
- **追手（ハンター）** はそのリンクを開くだけ。針と同じ**矢印と距離**が次の的を指し、
  順に巡ってゴールを目指す。**地図は出ない**——電波の薄い場所でも遊べる

針が「ひとつの場所へ戻る」道具なら、狐は「いくつもの場所を、隠して、巡らせる」遊びです。

## 通過のしるし、三つ

的ごとに、どうやって「通った」ことにするかを選べます。

| しるし | 狐がすること | 追手がすること |
| --- | --- | --- |
| 📍 **GPS** | 半径（10〜100m）を決める | その輪に入れば自動で通過 |
| 🔳 **QR** | アプリが作る QR を**印刷して現地に貼る** | **スマホの標準カメラ**で読むだけ（アプリ内スキャナ不要） |
| 📷 **写真** | なし | その場で一枚。写真が証明であり、ゴールの思い出になる |

**QR がズルできない理由**：QR が運ぶのは
`#t=コースID.番号.合言葉` というリンクの断片で、追手が読むとアプリが開いて通過判定します。
けれど狐が送る**コースのリンクには、合言葉そのものは入っていません**——
入っているのは合言葉の **SHA-256 ハッシュだけ**。リンクをいくら解読しても、
現地の QR を読まないかぎり、その的は通れません。順番も飛ばせません
（次の的のしるししか受け付けない）。

## サーバーがない、ということ

この遊びに**サーバーは一台もありません**。コースは URL のかけら（`#c=…`）に
Base64 で畳まれ、リンクを渡すことが、遊びを渡すことのすべて。狩りの進み具合も
写真も、**あなたの端末から一歩も出ません**。狐が QR を貼り、リンクを送り、
追手が歩く——その間、どこのサーバーも、誰の位置も記録しません。

## スマホでこそ

- **GPS** — 的を据えるときは数秒ぶんの測位を精度で重みづけ平均。歩くときは矢印に追従
- **磁気センサー** — 矢印が実際の向きを指す（`針`ゆずり。磁石が無ければ歩く向きで代替）
- **カメラ** — QR は標準カメラで読み、写真は端末内で縮めて保存（外に出さない）
- **共有シート** — コースも記録も、LINE やメールへそのまま
- **Wake Lock / 振動 / PWA** — 駆けている間は画面が落ちず、的に近づくと脈打ち、
  ホーム画面からオフラインで起動できる

## アーキテクチャ

```
works/kitsune/
├─ index.html · style.css · manifest · sw.js
└─ js/
   ├─ core/              ★ DOM 非依存
   │  ├─ geo.js          球面三角・磁気偏角・向きの平滑化（針と同じ心臓）
   │  ├─ course.js       コースとトークンの符号・通過の判定（Hunt）
   │  └─ qr.js           QR コードの符号化（Reed-Solomon・依存ゼロ自作）
   └─ ui/
      ├─ sensors.js · needle.js   感覚器と文字盤（針ゆずり）
      ├─ camera.js      写真を思い出の大きさに畳む
      └─ main.js        役の分かれ道・狐・追手
```

```bash
cd works/kitsune
node --test tests/*.test.js   # 24 tests
```

通過の判定・コースの符号・QR の構造は、すべてヘッドレスでテストされます
（CI は端末なしで、コースを畳んでほどき、QR を組んで**独立した読み手で読み戻し**、
狩りを最後まで歩かせて検めます）。QR は QR Code Model 2・バイトモード・
Reed-Solomon 誤り訂正を依存ゼロで自作したものです。

## 遊び方（はやわかり）

1. ひとりが「🦊 狐になる」。行きたい場所をいくつか歩いて、的を刺す。
   QR の的があれば「🖨 QRを刷る」で印刷して、こっそり貼る
2. 「📤 コースを送る」でリンクを仲間へ
3. 仲間はリンクを開いて「🦊 狩りをはじめる」。矢印を追って、的を巡る
4. 全部通れば、タイムと写真の記録が出る。それも送って自慢できる
