# 算 `san`

### a computer born from nothing — 無から生まれた計算機

ブラウザに住む架空の 16bit コンソール。CPU も、アセンブラも、言語も、<br>
画面も、音も、デバッガも、すべてここで生まれた。依存パッケージはゼロ。

## これは何

`算` は、**計算機を一そろい、無から作った**作品です。部品はぜんぶ自前です。

| 層 | 何があるか |
| --- | --- |
| **CPU** | 16bit・レジスタ 8 本・全命令 2 ワード固定長の独自命令セット（32 命令） |
| **メモリ** | 65536 ワード。VRAM もパレットもデバイスも、ぜんぶ番地の上にある |
| **画面** | 128×96 画素・16 色。既定のパレットは墨・胡粉・茜・藍……の伝統色 |
| **音** | 4 チャンネル（矩形・三角・鋸・雑音）。レジスタに書けば鳴る |
| **アセンブラ** | 2 パス。日本語ラベル可。誤りは行番号つきの日本語で返す |
| **珠** | 日本語キーワードの高級言語。`もし` `あいだ` `かぞえ` `関数` `かえす`——コンパイラがアセンブリに写す |
| **工房** | コード編集（自作ハイライト）・組んで走らせる・**ステップ実行・ブレークポイント・レジスタとメモリの覗き窓** |
| **ROM** | この機械の上で動くプログラムを 6 本同梱。**蛍が機械の中でもう一度光る** |

機械の唯一の真実は 2 枚の仕様書です——[HARDWARE.md](./HARDWARE.md)（機械）と
[LANGUAGE.md](./LANGUAGE.md)（言語）。コードはその写しにすぎません。

## なぜこれを作ったか

この作品集は、言葉（`言`）、歌（`歌`）、布（`織`）と、無からものが生まれる様を
作ってきました。けれどそれらはみな、**借り物の計算機**の上で走っています。
ブラウザという他人の機械の上で。

だから今回は、土台そのものを作りました。命令セットを決め、その命令だけで
動く CPU を書き、その CPU のための言葉（アセンブリ）を作り、その言葉が面倒に
なったので、もっと人に近い言葉（珠）と、それを翻訳するコンパイラを作る——
**計算機の歴史を、ひとつのディレクトリの中で繰り返す**。

そして作品集に住んでいた蛍を、自分の機械の中に放しました。
六月が終わっても、ここの蛍は消えません。

## 遊び方

`index.html` を開いて「電源を入れる」。

- **十字キー**＝矢印、**A**＝Z、**B**＝X、**開始**＝Enter、**選択**＝Shift
- ROM 棚から選ぶ：起動の画・蛍・へび（遊べる）・万華鏡・電卓（シリアルに話しかける）・恋歌
- **恋歌**だけは珠で書かれている。「歌」の seed 20260612 の世界で生まれた節
  「らドレドレドーそーー」を、「織」の染めの色で帯に積みながら、この機械が三角波で歌う。
  言葉（言）→ 音（歌）→ 紋様（織）と渡ってきた系譜が、ここで**機械の声**になる
- ソースはその場で書き換えられる。「組んで走らせる」で即、機械に入る
- 止めて、1 命令ずつ歩いて、レジスタとメモリを覗ける。行番号をクリックすれば
  ブレークポイント。**機械の心拍を見ながらプログラムが書ける**

## はじめての珠

```tama
変数 x = 64
変数 y = 48
くりかえし {
  塗る(0)
  もし ボタン() & 4 なら { x = x - 1 }
  もし ボタン() & 8 なら { x = x + 1 }
  点(x, y, 12 + (フレーム() >> 4) % 2)
  見せる()
}
```

はじめてのアセンブリ：

```asm
.const 口, 0xF006
  LDI r0, 口
  LDI r1, '無'
  ST [r0], r1     ; シリアルに「無」と書く
  HLT
```

## アーキテクチャ

```
works/san/
├─ HARDWARE.md           機械仕様書 — ISA・メモリマップ・デバイス
├─ LANGUAGE.md           言語仕様書 — 珠の文法と組み込み
├─ index.html · style.css
└─ js/
   ├─ core/              ★ DOM 非依存 — Node の中でも同じ機械が動く
   │  ├─ vm.js           CPU（命令の実行）
   │  ├─ bus.js          メモリ・VRAM・パレット
   │  ├─ devices.js      FLIP/FRAME・ボタン・乱数・打鍵・シリアル・音源
   │  ├─ asm.js          アセンブラと逆アセンブラ
   │  └─ machine.js      一式を箱に納めて、1 フレームの作法を持つ
   ├─ lang/              ★ 珠コンパイラ（字句 → 構文 → コード生成）
   ├─ ui/                画面・音・エディタ・デバッガ（ブラウザだけ）
   └─ roms.js · roms/    同梱 ROM 6 本（アセンブリ 5 本と、珠の恋歌）
```

すべてのテストはヘッドレスです。CI は毎回、**機械を起動し、コンパイラを通し、
全 ROM を何百フレームも実走させて**、画素単位・文字単位で検証します——
CPU 13・アセンブラ 13・デバイス 10・珠 45・ROM 24 の **105 tests**。

```bash
cd works/san
node --test tests/*.test.js
open index.html
```
