# 珠 — 算の高級言語

そろばんの珠（たま）から名前を取った、日本語キーワードの小さな言語。
コンパイラはアセンブリのテキストを吐き、`asm.js` がそれを語に写す。
この文書が言語のすべてである。ここに無い機能は、無い。

## 0. かたち

- 値はすべて **16bit ワード**（0–65535）。負数は 2 の補数として扱い、
  比較 `<` `<=` `>` `>=` は**符号つき**で行う。割り算 `/` `%` は**符号なし**。
- 文は改行で区切る。`{` `}` でブロックを作る。セミコロンは無い。
- コメントは `※` から行末まで（`#` でもよい）。
- 識別子：日本語・英数（数字始まり不可）。予約語は下記の太字。

## 1. 変数と式

```tama
変数 x = 10            ※ 宣言（初期値必須）
x = x + 1              ※ 代入
変数 y = (x * 3 + 偏り) % 256
```

- **変数** はいまのスコープに名前を作る。関数の外なら大域、内なら局所。
- 演算子（優先順位の高い順）：
  `!` `-`（単項） / `*` `/` `%` / `+` `-` / `<<` `>>` / `<` `<=` `>` `>=` / `==` `!=` / `&` / `^` / `|` / `&&` / `||`
- `&&` `||` は短絡しない（両辺を評価して 0/1 で合成する）。真は 1、偽は 0。
  条件の文脈では 0 以外すべて真。
- 数：`10`、`0x1F`、`0b1010`、`'あ'`（文字 → 符号単位）、`-3`

## 2. 制御

```tama
もし x > 5 なら {
  ...
} ちがえば もし x > 0 なら {
  ...
} ちがえば {
  ...
}

あいだ x < 10 {        ※ while
  x = x + 1
}

かぞえ i = 0 から 9 {   ※ for（両端を含む。上りのみ）
  点(i, i, 5)
}

くりかえし {            ※ 無限ループ
  もし 押した() & 64 なら { ぬける }
}
```

- **ぬける** はいちばん内のループを抜ける。**つづける** は次の周へ。

## 3. 関数

```tama
関数 足す(a, b) {
  かえす a + b
}
変数 z = 足す(3, 4)
```

- 引数はすべて値渡しのワード。**かえす** で値を返す（省略時 0）。
- 再帰できる。引数・局所変数はスタックに積まれる。
- 関数の中で関数は作れない。

## 4. 配列

```tama
配列 譜[16]            ※ 大域に 16 ワードの場所を取る（0 初期化）
譜[0] = 60
変数 v = 譜[i]
```

- 配列は大域のみ（関数内では宣言できない）。
- 添字の範囲は見張らない。この機械は親切ではない。
- 配列名を裸で使うと**先頭番地**になる（`おく(譜+1, 9)` のように使える）。

## 5. 組み込み（機械の体に触る）

| 呼び方 | すること |
| --- | --- |
| `点(x, y, 色)` | 画素を置く（0 ≤ 色 ≤ 15） |
| `色(x, y)` | 画素を読む |
| `塗る(色)` | 画面全部をその色で |
| `見せる()` | FLIP に書き、下りるまで待つ（= 1 フレーム） |
| `ボタン()` | BTN を読む（1=↑ 2=↓ 4=← 8=→ 16=A 32=B 64=開始 128=選択） |
| `押した()` | BTNP を読む |
| `乱数(n)` | 0 〜 n−1 の決定的乱数（n=0 は 0） |
| `種(v)` | 乱数の種を蒔く |
| `フレーム()` | FRAME を読む |
| `鍵()` | 打鍵キューから 1 文字（空なら 0） |
| `文字(c)` | シリアルに 1 文字書く |
| `表示(v)` | 数を十進で書き、改行する |
| `音(ch, 周波数, 音量, 波形)` | 音源レジスタ 4 つを一度に書く |
| `おく(番地, v)` | mem[番地] = v |
| `みる(番地)` | mem[番地] を読む |
| `止まる()` | HLT |

組み込みと同じ名前の変数・関数・配列は作れない。

## 6. コンパイラの API（実装の契約）

`js/lang/tama.js` が export するもの：

```js
compile(src) -> {
  ok: boolean,
  errors: [{ line, msg }],   // 日本語、行番号つき
  asm: string,               // 吐いたアセンブリ（ok のときだけ意味を持つ）
  words: Uint16Array,        // asm.js を通した語（ok のとき）
  origin: number,
  lineOf: (addr) => number,  // 機械語の番地 → 珠ソースの行番号（デバッガ用）
}
```

- 検出する誤り（すべて行番号つき日本語）：未宣言の変数 / 名前の重複 /
  引数の個数違い / ループ外の「ぬける・つづける」/ 関数外の「かえす」/
  構文の崩れ / 予約語の誤用
- 生成規約：式の評価は R0–R3 を使い、必要ならスタックへ退避。
  局所変数と引数は SP 相対。大域変数はラベルつきの .word。
  プログラムの末尾に HLT（落ちて隣の関数に入らないため）。

## 7. 例：蛍が一匹

```tama
変数 x = 64
変数 y = 48
くりかえし {
  塗る(0)
  もし ボタン() & 4 なら { x = x - 1 }
  もし ボタン() & 8 なら { x = x + 1 }
  点(x, y, 12 + (フレーム() >> 4) % 2)
  見せる()
}
```
