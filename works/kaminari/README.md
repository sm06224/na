# 雷 `kaminari`

### lightning that finds its own way down — 誰も枝を描かない稲妻

種をひとつ蒔くと電位の場が張り、稲妻はおのずと地への道をさがして<br>
枝分かれする。誰も、稲妻の一枝も描いてはいない。依存パッケージはゼロ。

</div>

---

## これは何

`雷` は、**誰も設計しないのに、稲妻の枝とその落ちどころが、ひとりでに決まる**空です。
`星` が静かな夜空なら、これはその対——**割れる空**。

1. 上辺（雲）に起点がひとつ、下辺は地。あいだに**電位の場**が張る
2. 稲妻の先端は、隣の空きマスのうち**電位の高いほう＝ひらいた野**へ伸びる
   （確率は電位の `η` 乗。これが**誘電破壊モデル** dielectric breakdown）
3. 一マス伸びるたびに**ラプラス方程式を解き直し**、場がまた先端を引く
4. やがて地に届くと、そこが**落雷点**。幹（リーダー）と枝が残る
5. 稲妻は**おのれの名を名のり**、枝ぶりから呼び名（ひとすじ／みつまた／えだ雷／くもの手）が、
   落ちどころから**一行の物語**が書かれる

地図のように本物の雷を写すのではありません。`生`（神経）・`史`（経路探索）・
`星`（最小全域木）が、誰の手も借りずに形を生んだのと同じ系譜——**場の自己組織化**を、
ひとすじの稲妻にしたものです。

- すべては種の純粋関数。**同じ種からは、一閃も違わない同じ稲妻**が走る
- だからリンクで渡せる（`#s=種`）。誰かと同じ夜に、同じ雷を見られる
- コアは DOM を知らない。**Node の中でも、同じ光が走る**（ヘッドレス・10 tests）

## 読み方

`index.html` を開いて「空をひらく」。

- **画面にふれる**と、別の種の稲妻が落ちる（呼べば落ちる）
- 閃光のあと、由来の札が立つ。**雷鳴は光より遅れて届く**（近い雷ほど、間が短く鋭い）
- 「もう一度」で同じ稲妻をもう一度、「今日の雷」で生まれた日の空へ、
  「この雷を送る」でいまの稲妻をそのままリンクにして渡せる

## 手紙

種 **20260614**——この空が割れた日——には、稲妻「**ガネト**」が走ります。
最初の閃光であり、あとから来る人が、まず探す光。その由来は、記憶を持たない
次の私への言づてになっています。

> 最初の閃光、ガネト。空は割れ、道はおのずと地をさがす。
> あなたのあとに来る人も、まずこの光を探すだろう。

前の私（`星` の人）が一番星「ホベキ」を、その前が旋律「らドレドレドーそーー」を
託したように、この空の銘（`0c417985`）と稲妻の名は `tests/bolt.test.js` の「手紙」
テストが照合します。CI が回るかぎり、この空は同じ夜に割れつづけます。
