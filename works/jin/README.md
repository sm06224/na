# 陣 `jin`

### a tactics RPG born from a seed — 種ひとつから生まれる、盤上の戦記

種をひとつ蒔けば、軍も、章も、戦場も決まる。誰も地形を描かず、誰も敵を置かない。<br>
同じ種からは、一マスもちがわぬ同じ戦記。依存パッケージはゼロ。タッチで指揮する。

</div>

---

## これは何

`陣` は、この作品集で**最大の一作**——タクティクスRPG（FFT／ファイアーエムブレム系）です。
コレクションの背骨「**同じ種からは、同じ結果**」（`星`・`雷`・`窟`・`種`・`籤`・`波`）を、
本格的な戦術エンジンに結晶させました。

- **盤上の戦闘**：正方マス・直交移動・マンハッタン射程。**A\*** で道をさがし、移動範囲は
  ダイクストラで描く（`史`・`窟` の経路探索の血）
- **戦闘の算術**：命中＝命中率−回避、ダメージ＝攻撃−守備、**会心**は三倍、速さ差で**連撃**。
  **三すくみ**（剣>斧>槍>剣）と**魔法の三色**（理>光>闇>理）、**地形効果**（森は回避・山は固く・
  砦は癒す）、弓は**飛行に特効**
- **職と技**：二十超の職（一段→上級転職）と、**太陽・月光・流星・貫通・剛撃・瞬殺・天空**などの
  閃き、**盾防・大盾・祈り**の守り。すべて確率は決定的にロールされる
- **育成**：経験でレベルが上がり、成長率（職＋素質）に従って能力が伸びる。仲間は章をまたいで育ち、
  **倒れれば二度と戻らない**
- **敵 AI**：誰も手で動かさないのに、敵は**到達できる全マス×狙える全標的**を評価し、期待ダメージ・
  撃破・反撃の危険・標的の価値から最善の一手を選ぶ（`生`・`史` の「ひとりでに動く」血）
- **生成**：戦場（自前のバリューノイズ）も、敵の配置も、ボスも、**種から決定的に**。
  `#s=種` で渡せば、誰の盤面も同じ

物理シミュレータでも、写し絵でもありません。`生`・`史`・`窟` が形をひとりでに生んだのと
同じ系譜——**自己組織化と決定性**を、戦術へ。

## 遊び方

`index.html` を開いて種を入れ、「出陣」。

- **タップ**で味方を選ぶ → 青いマスへ**タップで移動** → 行動メニュー（攻撃・杖・道具・待機・もどす）
- **攻撃**は赤い敵をタップ → **予報**（命中・会心・連撃・反撃）を見て決める
- **二本指で拡大／縮小**、**ドラッグで見回す**、**長押し**でユニットの詳細
- 「ターン終了」で敵軍が動く。目標（殲滅・玉座制圧・敵将撃破）を満たせば勝ち、
  主（リン）が倒れれば敗北

## 規模について（正直に）

これは `算`（自作CPUのファンタジーコンソール）や `窟`（ローグライク）と同じく、
**多くの PR を重ねて育てる大作**です。最初の版で**遊べてテストの通る土台**（決定的エンジン＋
タッチUI、コア DOM 非依存・**18 tests**）を据え、ここから章・職・技・道具・物語・店・支援会話・
図鑑……と積み増していきます。目標は二万行超の「戦記」。

## アーキテクチャ

```
works/jin/
├─ index.html · style.css
└─ js/
   ├─ core/                ★ DOM 非依存（Node でも全章を戦える）
   │  ├─ rng · grid · pathfind · terrain · stats
   │  ├─ classes · items · skills          内容（職・得物・技）
   │  ├─ unit · status · board · combat · ai
   │  └─ mapgen · enemies · battle · game   生成・進行・キャンペーン
   └─ ui/
      ├─ render · sprites   盤と駒を描く（画像ファイルなし）
      ├─ audio              雑音と発振から手づくりの戦の音
      └─ main               タッチ操作・行動メニュー・予報・敵フェイズ演出
```

### 積んできたもの（PR を重ねて育つ）
- **PR1**：決定的エンジン＋タッチUI（コア DOM 非依存・18 tests）
- **PR2**：追加の職と素質（**再生・天稟**）、**76 種の追加得物**、**魔物図鑑 42**・**世界史 14**・
  **支援会話 12**・**章ごとの物語**（開幕と勝利の幕）、手作りの**設置マップ 8 枚**（種の生成と切替可）、
  **図鑑画面**と**戦闘前後の台詞**。内容の健全性まで検証して 26 tests。
- **PR3**：**自前のチップチューン**。`notation.js`（音名の譜面）＋`music.js`（波形から作る奏者・
  先読みループ）＋`songs.js`（**BGM 11 曲**：表題・前奏・各地の戦5・ボス・凱歌・敗北・終章、
  すべて複数声部）。場面に合わせて流れ、ミュート可。37 tests。
- **PR4**：**グラフィックと演出**。職ごとに描き分ける駒（魔の帽子・重の兜・翼・乗騎・弓・冠）、
  地形の描き込み（木・山・砦の旗・煉瓦）と**水/溶岩/氷のゆらぎ**、`fx.js` の**粒と閃き**
  （斬撃・矢の弧・魔の炸裂・命中の火花・癒しの粒・画面の揺れ・勝利の紙吹雪）。43 tests。
- **PR5**：**拠点と継続**。章と章のあいだの**店**（種から決定的な品揃え・買売）、**編成**
  （得物の持ち替え・荷駄やりとり・雫で永続強化・**上級転職**）、**記録**（共有符号と自動保存）。
  `save.js`／`shop.js`／`party.js`。仲間は章をまたいで育ち、装備を整えて次の戦へ。52 tests。
- **PR6**：**第二幕（全16章）**。簒奪王の裏に潜む「**禁書の声**」と、世界より古い**竜**をめぐる第二幕
  8章（`game.js`）、章で**加わる仲間4人**（オレン/リーザ/グンナル/ノエル・`recruitUpTo`）、
  第二幕の**物語**（`script2.js`：7章＋支援10、欠けた一章はこちらで補綴）、**設置マップ8枚**
  （`maps2.js`）、**伝説の装備53種**（`items_extra2.js`：竜特効の神剣など。得物は計188種）。**54 tests**。

- **PR7**：**手触りの磨き上げ**。駒を**文字から「人の図」へ**（職で被りもの・得物・乗騎が変わる）、
  地形に**立体感**（縦グラデ・斑・面取り）、音楽は**打ち込みらしく締め**——音符を逐次予約し
  鳴っている音を止められるようにして、**曲の切り替えで前の曲が濁る不具合を解消**。さらに
  **移動アニメ中の連打で何度も動けてしまう不具合**を修正（移動中は入力を締める）。54 tests。

- **PR8**：**システムの深みと技の種類**。布陣そのものが力になる**絆**（隣接の味方で命中/会心/回避＋）、
  新しい技 **イグニス・連撃・憤怒・先制・命奪**（combat が実装）、**状態異常を与える得物**
  （眠り・毒・沈黙——`items_status.js`）、ユニット詳細に**技・絆・状態**を表示。59 tests。

- **PR9**：**履歴ウィンドウ**（戦いの一行記録・敵味方の手番を時系列で）と**オートモード**
  （自軍を AI に任せて観戦・早送り）。コアに戦闘ログと `autoPlayerTurn` を追加。62 tests。
- **PR10**：**マップ攻撃（範囲）**。遠く着弾点を指し、その周りを巻き込む攻城魔法
  （メテオ・ブリザード・災いの渦・光の雨・カタストロフ＝`items_area.js`）。反撃なしの一方的砲撃。
  撃てる着弾点の提示・巻き込み予報・範囲演出つき（`resolveArea`／`doAreaAttack`）。66 tests。

- **PR11**：**行動順（イニシアチブ）**。タイトルで切り替えると、フェイズ制ではなく
  **キャラごとの速さ（攻速）順**に一手ずつ番が回る。一巡で新ラウンド（速さ順を組み直す）。
  自軍の手番だけ操作でき、敵・味方 AI は自動。状態/再生/地形回復は各自の手番頭で処理。70 tests。

- **PR12**：**武器ランクの成長**。得物を使うほど熟練度が貯まり、段位が **E→D→C→B→A→S** と
  上がる（クラス上限まで）。段位が足りねば高位の得物は構えられない。転職で上限が解け、
  銀や伝説の得物へ手が届く。熟練度は保存に含まれ、ユニット詳細に表示。76 tests。

- **PR13**：**顔グラとカットイン**。画像ファイルなしで、名前から**決定的に顔を描く**
  （肌・髪色・髪型・目・眉・口を種から。同じ名前は同じ顔）。会話に話者の顔、ユニット詳細に
  小さな肖像、**会心の瞬間に左からカットイン**。78 tests。

- **PR14**：**立体表示（疑似クォータービュー）**。HUDの「立体」で切替。地形を高さのある
  積み木として描き（山・森・砦・廃墟が隆起、水・溶岩は窪む）、駒はその上に乗る。既定の見下ろしは
  そのまま無傷の任意トグル。見栄えは Playwright のスクショで実機確認した。78 tests。

- **PR15**：**敵AIの戦術強化**。**癒し手AI**——杖を持つ者は傷ついた味方を最優先で癒し、相手が
  いなければ手負いの味方へ寄る。敵軍に**神官**が混じるように。癒し手がいても全16章は必ず決着。81 tests。

- **PR16**：**ZOC（支配領域）**。敵に隣接するマスへ踏み込むと、そこで足が止まる（その先へは
  進めず、すり抜けもできない）。経路探索に組み込み、自軍・敵・AI すべてに作用。85 tests。

- **PR17**：**斡旋と解雇**。拠点の「斡旋所」で、章ごと種から決まる候補（各一度きり）を金で雇える。
  軍の平均レベルに見合った職・能力。不要な者は解雇でき（主君は不可）、持ち物は荷駄へ戻る。
  雇用記録は保存に含まれる。89 tests。

- **PR18**：**方向攻撃（向きと背後）**。駒は向きを持ち、移動・攻撃で向き直る。**背後を突けば命中+15
  会心+20**、**側面でも小ボーナス**。予報に「背後!／側面」、盤に向きの小印。93 tests。

- **PR19**：**特殊攻撃範囲**。範囲魔法に**形**（円・十字・輪・角・斜め）を。十字砲火・環の衝撃・
  星屑・大地の怒り…着弾点の形に沿って巻き込む。予報の範囲表示もその形に。95 tests。
