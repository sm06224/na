# 奏 `kanade`

### an instrument for everyone at once — みんなで触れる、光の楽器

ひとつの画面に、何本でも指を。誰がどこを触っても音は濁らない。<br>
弾いた音と光は、八秒たつと、すこし薄くなって還ってくる。依存パッケージはゼロ。

## これは何

`奏` は、**音楽の素養がなくても、その場の全員で合奏できる**楽器です。
スマホやタブレットを輪の真ん中に置いて、みんなで触ってください。

- **触れると鳴る**。上ほど高く、左右はそのまま音の定位に。なぞれば走句になる
- **マルチタッチ**＝そのまま合奏。デスクトップなら**キーボード**（A〜; が低い列、Q〜P がその 1 オクターブ上）でもうひとり弾ける
- **誰が弾いても濁らない**：使える音を**半音を含まない五音音階**（アンヘミトニック・ペンタトニック）に量子化してある。半音のぶつかりが存在しないので、でたらめに触っても必ず協和する
- **こだま**：弾いた音と光は 8 秒後に少し薄くなって還ってくる。ひとりなら重ね録りに、
  数人なら模様が編み上がっていく。巡るたび薄れて、やがて静かに消える
- **調べ**：主音と五度のごく薄い持続音が、場をひとつの調に保つ
- 残響はホワイトノイズから手作りしたインパルス応答。**音源ファイルなし・サーバーなし・登録なし**

## 四つの音階

| 音階 | 中身 | 気分 |
| --- | --- | --- |
| **宵** | メジャーペンタトニック | あたたかい |
| **雨** | マイナーペンタトニック | しっとり |
| **風** | 律音階（雅楽） | 風がとおる |
| **凪** | 宙づりの音階 | どこにも帰らず、ただ漂う |

琉球音階や平調子も検討しましたが、半音を含むため「誰が触っても濁らない」という
この楽器の約束に反するので外しました（この約束はテストが守っています）。

## アーキテクチャ

```
works/kanade/
├─ index.html · style.css
└─ js/
   ├─ core/music.js     ★ DOM 非依存 — 音階・量子化・こだまの論理
   ├─ ui/audio.js       Web Audio — つまびき・残響・調べ
   └─ ui/main.js        指と鍵盤と光
```

`庭`（環境音の庭）と `歌`（創発する音楽）の血を引いています。庭は眺めるもの、
歌は世界が勝手に歌うもの——`奏` は、**あなたたちが鳴らすもの**です。

## 検証

```bash
cd works/kanade
node --test tests/*.test.js   # 12 tests
```

テストは「全音階に半音のぶつかりがない」（協和の保証）、「5 段上はちょうど 1 オクターブ」、
「こだまは巡るたび薄れ、やがて忘れられ、取りこぼしても重複しない」ことを固定します。
