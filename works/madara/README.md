# 斑 `madara`

### the coats that pattern themselves, born from nothing — 一様な卵が、どうして豹の斑になるのか

豹の斑も、虎の縞も、誰かが描いた模様ではない。<br>
ふたつの物質が追いかけあうだけで、無地の肌にひとりでに浮かぶ——その物理を、種ひとつから。依存ゼロ。

## これは何

`斑` は、**反応拡散**——生きものの肌に、斑・縞・迷路・孔がおのずと立つ仕組み——を、模様を一筆も描かずに**物理から**生やす作品です。

アラン・チューリングが**最後の論文**（1952, "The Chemical Basis of Morphogenesis"）で問うたのは、この謎でした。受精卵は一様なのに、どうして豹は斑に、虎は縞になるのか。彼の答えは、たった四則の連立でした。

```
U（餌）  はどこにでもあり、たえず補充される（feed）
V（喰うもの）は U をふたつ喰って自分をふやす（U + 2V → 3V）
ふたつは別の速さで滲み（拡散し）、V は一定の率で枯れる（kill）
```

ここで肝心なのは、**U のほうが V より速く滲む**こと。喰うものより餌が速く広がるから、V は「少し離れた所では餓えて死に、近くでは栄える」——この近場の活性化と遠方の抑制だけで、対称な無地が破れ、模様が立ちます（チューリング不安定）。本作は Gray–Scott の式でこれを解きます。模様を選ぶつまみは、たった二つの数 `f`（feed）と `k`（kill）だけ。

### 二つの数が、肌の運命を決める

`(f, k)` の値ひとつで、立つ模様が変わります。`madara.js --map` で、その**位相空間の地図**が読めます——どこに斑が、どこに縞が、迷路が、孔が棲むか。

| 肌 | あらわれ | 棲む獣 |
|---|---|---|
| **斑** spots | 孤立した粒が散る | ヒョウ・ジャガー・キリン・テントウムシ |
| **縞** stripes | 帯がのびる | トラ・シマウマ・エンゼルフィッシュ |
| **迷路** maze | 筋がつながり地を半ば占める | ノウサンゴ・ハコフグ・指紋 |
| **孔** holes | 地が満ち、ぽつぽつ孔があく | ハチの巣・海綿・蓮の実 |

### ひとりでに名のる

育ちきった肌を、本作は**形から正直に測ります**（覆い率・島の数・隙間の数）。生まれた土地（バイオーム）が何であれ、肌に実際に立った模様を見て分類し、**棲む獣を一頭、種から名のらせる**。豹は斑から、虎は縞から生まれる——coat に偽りはありません。`言`・`星`・`雪` と続いた「ひとりでに名づける」系譜の、肌の章です。

- コアは **DOM もネットワークも知らない**。同じ種からは寸分たがわず同じ肌（決定的）
- 物理の正しさは**ヘッドレス検証・18 tests**：ラプラシアンは一様な地を滲ませても一様（重みの総和 0）、U は V より速く滲む、無地は無地のまま（不動点）、ひと粒の種火が肌いちめんに広がる（創発）、そして `(f,k)` は約束どおり斑・縞・迷路・孔を生む——すべて数で確かめてあります

## 遊び方

**ブラウザ**：`index.html` を開く。無地の肌に種火が落ち、模様が育つ。**なでる**と種火が撒かれ、模様が割れて、またひとりでに結ぶ。育ちきると、肌は自分が誰の毛皮かを名のる。鍵（`#m=`）を分かち合えば、同じ肌が同じように育つ。

**ターミナル**（絵が見えなくても、本物の反応拡散が走る）：
```bash
node madara.js                  # その日の肌を一枚（端末に 24bit カラーで）
node madara.js <種>             # その種の肌
node madara.js <種> --steps 8000   # もっと育てる
node madara.js --map            # (f,k) 位相空間の地図（どこにどの肌が棲むか）
```
