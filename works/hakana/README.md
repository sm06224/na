# 儚 `hakana`

### the fleeting colours of a thin film, born from nothing — 消える直前の、いちばん美しい色

## これは何

`儚` は、**薄膜の干渉色**——シャボン玉や油膜が虹いろに見える、あの色——を、絵の具を一切使わずに**物理から**描く作品です。

膜の表で跳ねかえった光と、裏まで潜って跳ねかえった光は、わずかにずれて重なります。膜の厚みが波長の整数倍にあたる色は弱めあって消え、半端な色は強めあって際立つ。だから**厚みが、色になる**。ここには色を塗る場所がありません。あるのは、波長と、膜の厚みと、人の目の感度だけです。

```
膜の厚み → 表裏の反射の干渉（Fresnel＋Airy）→ 残ったスペクトル R(λ)
        → CIE 1931 等色関数で目の色 XYZ へ → D65 の白のもとで sRGB
```

### なぜ、ただ計算するだけで美しいのか

自然が美しいからです。正しく光学を解けば、出てくる色の並びは Newton が数えた順そのまま——**黒 → 銀白 → 金 → 紅紫 → 青 → 緑 …** と、厚みが増すごとに次数を巡ります（`--scale` で見られます）。想像で「綺麗な色」を選んではいません。色は、物理の結論です。

### 生きていて、儚い

膜はとどまりません。

- **重力**で水が下へ落ち、膜は上から薄くなる
- **マランゴニ対流**でゆっくり渦を巻く
- やがて上に**黒い膜**が口をあける——厚みがほぼ 0 になり、表裏の反射が完全に打ち消しあう場所。これは破れの予兆です
- そして、**いちばん濃い色が出るのは、消える直前**

`儚（はかな）い`——この作品の名は、そこから採りました。最も美しいものが、最も早く消える。シャボン玉を見たことのある人なら、知っているはずの哀しさです。

- コアは **DOM もネットワークも知らない**。同じ種からは寸分たがわず同じ膜・同じうつろい（決定的）
- 物理の正しさは**ヘッドレス検証・9 tests**：白は (1,1,1) に、単色は正しい原色に、厚み 0 は黒い膜に、λ/4 で強めあい λ/2 で弱めあい、厚みが増せば色は次数を巡り、膜は上から薄り黒い膜を育てる——すべて数で確かめてあります

## 遊び方

**ブラウザ**：`index.html` を開く。膜が張り、虹がめぐる。**なでる**と膜が寄って色がにじみ、**ふれる**と破れて、また新しい膜が張る。「別の膜」で次の種へ。鍵（`#h=`）を分かち合えば、同じ膜が同じように消えていく。

**ターミナル**（絵が見えなくても、本物の色が出る）：
```bash
node hakana.js              # その日の膜を一枚（端末に 24bit カラーで）
node hakana.js <種>         # その種の膜
node hakana.js <種> --t 9   # 9 秒後の姿（黒い膜が育つころ）
node hakana.js --scale      # 干渉色の早見表（Newton の色階）
```

## アーキテクチャ

```
works/hakana/
├─ index.html · style.css        膜を張り、なで、ふれて破る（キャンバス＋ブルーム）
├─ hakana.js                     CLI（端末に干渉色を描く／色階を示す）
└─ js/
   ├─ core/spectrum.js           ★ スペクトル→目の色：CIE 1931 等色関数（解析近似）＋D65→sRGB
   ├─ core/film.js               ★ 薄膜の干渉：Fresnel＋Airy で反射率 R(λ)・干渉色の早見表
   ├─ core/flow.js               ★ 生きている膜：水切れ・マランゴニ・黒い膜（種で決まる）
   ├─ core/render.js             ★ 厚み場→ピクセル（ドームの曲率で入射角を変える）
   └─ ui/main.js                 低解像で描き、なめらかに引き伸ばし、ほのかに発光させる
```
