# 備 `sonae`

### 我が家の備蓄は、三分でわかる — 家庭の防災備蓄チェッカー

家族構成を入れると、必要な備蓄の品目と数量がすぐに出る。<br>
チェックすると「備え率」が見え、A4 のチェックリストとして印刷できる。<br>
無料・登録不要・オフラインで動き、**データは端末から一歩も出ない**。依存パッケージはゼロ。

</div>

---

## これは何

「水って何リットルいるんだっけ」を、三分で終わらせる道具です。

- **家族構成**（大人・子ども・乳幼児・高齢の家族・生理用品を使う人・ペット）と
  **日数**（最低限の 3 日／推奨の 7 日）を入れるだけ
- 水・食料からトイレ・あかり・乳幼児用品・ペットフードまで、**約 30 品目**を数量つきで算出
- 各品目に「なぜ要るのか」の一言つき。チェックすると**備え率**が上がっていく
- **🖨 印刷**すると、冷蔵庫に貼れる A4 チェックリストになる（チェックボックス付き）
- **テキストコピー**で家族のチャットにそのまま貼れる
- 入力とチェックは `localStorage`（この端末）にだけ保存。サーバーは存在しない

## 数量の根拠

| 目安 | 出典 |
| --- | --- |
| 水 1人1日 3L（飲料2L＋調理1L）、最低3日・推奨1週間 | 農林水産省「災害時に備えた食品ストックガイド」・内閣府 |
| 主食 1人1日3食、簡易トイレ 1人1日5回 | 東京都「東京備蓄ナビ」の算定の考え方 |
| カセットボンベ 1人1週間 約6本 | 同上（実測では下回ることも。本文に注記） |
| ペットのフード・水 最低5日分・できれば7日分以上 | 環境省「人とペットの災害対策ガイドライン」 |
| 乳幼児のミルク・離乳食 7日分、おむつは発災後に品薄化 | 自治体の備蓄啓発資料 |

あくまで一般的な目安です。持病・アレルギー・体格・住環境に合わせて増減してください。

## なぜ作ったか

東京都の「東京備蓄ナビ」は良い道具ですが、オンライン前提で、チェックの記録は残りません。
災害への備えはリストを**一度出して終わり**ではなく、**埋めていく過程**です。
この道具は、出したリストにチェックがつけられて、印刷して貼れて、次に開いたとき続きから始められます。
そして防災ツールこそ、**ネットが无くても動く**べきです（このページは一度開けばオフラインでも計算できます）。

`番`（シフト表）に続く、作品集で二番目の実用品です。

