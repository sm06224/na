# 群 `mure`

### a murmuration that has no leader, born from nothing — 指揮者のいない、夕暮れのむれ

何百羽の椋鳥が、ひとつの体のようにうねる。けれど率いている鳥は一羽もいない。<br>
一羽が見るのは近くの数羽だけ——その三つの約束だけで、空がうねる。依存ゼロ。

</div>

---

## これは何

`群` は、**椋鳥のむれ**——夕暮れに空をうねる、あの黒い波——を、誰も振り付けずに**規則だけ**から立ちあげる作品です。

一羽の鳥が見ているのは、半径いくらかの内側にいる**近くの数羽**だけ。その仲間に対して、三つの約束を守ります。

```
整列（align）   ：近くの仲間と、向きをそろえる
結束（cohere）  ：近くの仲間の、真ん中へ寄る
分離（separate）：近すぎる仲間からは、離れる
```

たったこれだけ。中央の指揮者も、全体の設計図もありません。なのに、何百羽が**ひとつの体のように**うねる——これが**創発**です（Reynolds の boids）。整列の度合い（order parameter）は、ばらばらの始まり（≈0）から、見るまにひとつ（≈1）へ立ちあがります。

### 隼がよぎると、波が走る

むれに**隼**が近づくと、近くの鳥がおびえて逃げ、その**おびえが仲間へ伝わって**いきます。鳥そのものより速く群れを走る、あの「驚きの波（agitation wave）」。むれは割れ、渦を巻き、隼が去れば**またひとつに結び直します**。

- コアは **DOM もネットワークも知らない**。同じ種からは寸分たがわず同じうねり（決定的）
- 「ひとりでに」を理に保証する**ヘッドレス検証・7 tests**：同じ種は同じうねり／むれは空に留まり破綻しない／ばらばらから向きが揃う（創発）／近すぎず離れすぎず（密度）／隼がよぎれば散りおびえが灯る／去ればまた結ぶ——すべて数で確かめてあります

`生`（人工生命）・`言`（言葉）・`歌`（音楽）と続いた「**ひとりでに生まれる**」系譜の、夕の章です。

## 遊び方

**ブラウザ**：`index.html` を開く。夕空にむれが放たれ、うねりはじめる。**触れると、あなたが隼になる**——むれは散り、波を立て、指を離せばまた結ぶ。「♪ 夕の音」で、低いドローンと、おびえで増す風が鳴る。「別のむれ」で次の種へ。鍵（`#s=`）を分かち合えば、同じ空を同じようにうねらせられます。

**ターミナル**（絵が見えなくても、本物のうねりが立つ）：
```bash
node mure.js              # その日のむれを一枚（端末に夕空の影で）
node mure.js <種>         # その種のむれ
node mure.js <種> --steps 800   # もっと飛ばす
node mure.js <種> --frames 90   # うねりを動かす（途中で隼がよぎる）
```
