## これは何

`謎` は、第二次大戦の暗号機 **Enigma** を、無から組み直したものです。鍵を押すと電流が
プラグボード → 三つのローター → 反射器 → ローターを逆順に → プラグボードと巡り、ランプが灯る。
`証 akashi`（現代の一方向ハッシュ）と対をなす、**歴史の、巡って戻る暗号**の章。

- **ローター I–V**・**反射器 B/C**・**リング設定（Ringstellung）**・**プラグボード**
- 史実の**ダブルステッピング**（爪機構の癖で、中央ローターが連続して進む）まで再現
- **暗号化と復号は同じ操作**（可逆）。同じ設定でもう一度通せば、文は元へ戻る
- **どの文字も、自分自身には化けない**——Enigma の宿命の弱点（連合国はここを突いた）

### 正しさは、史実が保証する

機械の正しさは、想像ではなく**公開されたテストベクタ**で確かめられます。

```
I-II-III / 反射器 B / リング AAA / 位置 AAA / プラグ無し
  AAAAA            → BDZGO
  A×25             → BDZGOWCXLTKSBTMCDLPBMUQOF
```

- コアは **DOM もネットワークも知らない**。同じ鍵からは、寸分たがわず同じ暗号文（決定的）
- ベクタ一致・可逆・自己無写像・ダブルステッピング・配線の健全性まで**ヘッドレス検証・10 tests**

### 鍵は、会える種

> 鍵（設定）を分かち合えば、**同じ機械を組んだ者だけ**が文を開く。一文字でも違えば、永遠にノイズ。

鍵は一本の文字列に畳めます（`#k=` で渡せる）：

```
ロータ-ロータ-ロータ.反射器.リング.初期位置.プラグ
例： I-II-III.B.AAA.AAA.AB CD
```

## 使い方

**ブラウザ**：`index.html` を開く。ローターを回し、鍵盤を打つと、ランプが灯ってテープに積もる。
「鍵を振る」で無作為な設定、「分かち合う」で鍵へのリンク。

**ターミナル**（成果物をそのまま）：
```bash
node nazo.js --key "II-IV-V.B.AHJ.XYZ.AB CD EF" "ATTACK AT DAWN"   # → 暗号文
echo "BDZGO" | node nazo.js                                        # 既定の鍵で復号
```
暗号化したものを同じ鍵でもう一度通せば、平文に戻ります。
