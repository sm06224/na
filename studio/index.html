gantt

製品リリース計画

日程・依存・進捗のガントチャート。横ドラッグで開始日、縦ドラッグで行の並び。依存は矢印で結ばれ、週末と today 線が入る。

ひらく → arch

サービス機能構成図

ノード・エッジ・グループのアーキ図。位置を書かなければ依存の深さで自動段組み。ドラッグで自由に配置できる。

ひらく →

書き方（DSL）

kind gantt
title 製品リリース計画
start 2026-07-01
section 設計
  task req    "要件定義"  at 2026-07-01 for 5d done 100
  task design "基本設計"  after req for 7d
  milestone ship "出荷"   after design

# @layout トレイラだけに書き戻り、上の意味部は汚れない。
